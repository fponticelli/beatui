Available Breakpoint Prefixes

All utility classes can be prefixed with these breakpoints:

  • bu-sm: - Small screens (≥40rem / 640px)
  • bu-md: - Medium screens (≥48rem / 768px)
  • bu-lg: - Large screens (≥64rem / 1024px)
  • bu-xl: - Extra large screens (≥80rem / 1280px)
  • bu-2xl: - 2X large screens (≥96rem / 1536px)

Example Usage:

<div class="bu-flex-col bu-md:flex-row bu-gap-2 bu-lg:gap-4">
  <div class="bu-w-full bu-md:w-fit">Content</div>
</div>
